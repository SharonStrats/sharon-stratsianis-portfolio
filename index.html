SolidOS Open Source

SolidOS is an Operating System for Linked Data. Solid and Linked Data are two important technologies that will allow us to have more control over our own data, hopefully in the near future. The SolidOS system allows multiple applications to exist in one Operating System all using the same data in different ways and for different reasons. I am working to improve the code base. When you first learn about this it can feel a bit overwhelming to imagine a future where you can interact with your data all from one location and one login. If you are interested, Digita have some great videos explaining it all here

Multiplication Game

This is a realtime multiplication game that allows children to play with other children to help learn their multiplication tables. The game displays a problem and gives a few options and the child needs to select the correct answer. The game uses the Suzuki Kasami Distributed Algorithm to pass a token around at the end of the game to give access to the critical section of determining who wins. If all children had the same result the child who pressed the Play Button first would win the game.

Built with: React, Suzuki Kasami, Socket.io

Sigmafy

Sigmafy is an application built on the Solid Node Server. It uses the Solid specification and Linked Data concepts to see how we can have fine-grained access control to a users location. A user's location gets stored at three different specificity levels and the user can control which app and people get access to a particular specificity.

Built with: React, Solid, Linked Data
